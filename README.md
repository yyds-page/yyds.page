# yyds.page

## Start

1. clone 
```shell
git clone -b main --recurse-submodules https://github.com/yyds-page/yyds.page.git 
```

2. install hugo
```shell
brew install hugo
```
see more [hugo quickstart](https://gohugo.io/getting-started/quick-start/)

3. live build
```shell
hugo server
```

visit: http://localhost:1313/
